# 🎸 BANDURA Project

**BANDURA** — це десктопний застосунок для розпізнавання музики. Проєкт розроблено в рамках курсової роботи з дисциплін КПІ та ОП

## 📌 Опис проєкту
Головна мета програми — захоплення аудіо з пристроїв відтворення комп'ютера (наприклад, навушників чи динаміків) та швидке розпізнавання треку. 

Застосунок має гібридну архітектуру:


## 🛠 Технологічний стек
* **Мови програмування:** Python 3.10+
* **Бібліотеки Python:** `shazamio`, `soundcard`, `soundfile`, `asyncio`, `pystray`, `Pillow`
* **Середовище розробки:**  PyCharm
